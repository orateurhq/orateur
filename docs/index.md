# Orateur

Minimal local **speech-to-text** (STT), **text-to-speech** (TTS), and **speech-to-speech** (STS) assistant, with optional MCP tools and a desktop app.

See the [repository README](https://github.com/orateurhq/orateur) for installation, GPU setup, and usage.
